- FYI, data 区域的列为
  (1) code
  - 可能的值为 P FI RT SC I H W PI

    - P  = Purchase
    - FI = Free Issued Mat'l
    - RT = Record to Manufacture
    - SC = Subcontract
    - I  = Inspect
    - H  = Hold
    - W  = Witness
    - PI = Packaging Inspection
  (2) type
  - 标注了当前 process 的类别
  (3) link
  - 后续指定具体功能, 目前仅作为介绍

  - Comb 行号 标注当前行与指定行形成 combo, 即如果本行被选择, 那么指定行也会同时出现在当前行下面
  - Hint 行号 标注当前行与指定行形成 hint, 即如果本行被选择, 那么指定行会作为 hint 提示在下方显示
  (4) process
  - 过程的具体内容
  - 可能包含类似 python 的占位符 {1}
  - 此类占位符提示该处存在由用户决定的具体字符串
